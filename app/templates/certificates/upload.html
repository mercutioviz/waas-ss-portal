Upload Tips
PEM Format
  • Upload the certificate file (.pem or .crt)
  • Upload the private key file separately (.pem or .key)
  • Include intermediate certificates in the certificate file if needed
PFX/P12 Format
  • Upload the .pfx or .p12 file as the certificate
  • No separate key file is needed
  • Provide the PFX password if the file is password-protected
